# Dataset Instructions（前処理ガイド）

> コンペ公式の翻字・翻訳テキストのフォーマット処理に関する指針

## 翻字テキストの主要課題

### 1. フォーマットの複雑さ
標準的な翻字フォーマット（ハイフン区切り音節）に加え、上付き文字・下付き文字・アッシリア学専門の句読点が混在。

参考: [Complete Transliteration Conversion Guide](http://oracc.museum.upenn.edu/doc/help/editinginatf/primer/inlinetutorial/index.html#Compound)

### 2. 大文字化（Capitalization）
- **先頭大文字**: 固有名詞（人名・地名）
- **全大文字（ALL CAPS）**: シュメール語のロゴグラム（書記の簡略表記）

### 3. 限定詞（Determinatives）
名詞・固有名詞の分類子。波括弧 `{}` で囲まれる（翻字テキスト中で波括弧はこの用途のみ）。

例: `a-lim{ki}`, `A-mur-{d}UTU`

### 4. タブレットの破損
古代テキストのため破損・欠落あり。以下に統一を推奨:
- `<gap>`: 1文字の小さな破損
- `<big_gap>`: 複数文字～大きな破損

## 翻訳テキストの課題

- 古アッシリア語テキストの完全な翻訳データベースは存在しない
- 書籍・論文からOCR + LLM補正で収集 → フォーマットの不統合が残る
- 固有名詞の大文字化はMLで最もパフォーマンスが低下する箇所
- レキシコン（辞書）に正規化済み固有名詞を収録

## 現代写字記号（Modern Scribal Notations）

### 行番号
- 通常: 1, 5, 10, 15, ...
- 破損行: `'` 付き（例: `5'`）
- 二重破損: `''` 付き（例: `5''`）← 引用符ではない

### その他の記号

| 記号 | 意味 |
|------|------|
| `!` | 困難な読みについて確信あり |
| `?` | 困難な読みについて不確定 |
| `/` | 行の下に属する文字 |
| `:` または `.` | 古アッシリア語の語分割記号 |
| `( )` | 破損や消去のコメント |
| `< >` | 書記の挿入・修正 |
| `<< >>` | 誤った・誤記の文字 |
| `˹ ˺` | 部分的に破損した文字（半括弧） |
| `[ ]` | 明確に破損した文字・行 |
| `{ }` | 限定詞（Determinatives） |

## 前処理推奨（Remove）

以下の現代写字記号を **除去** する:

| 対象 | 説明 |
|------|------|
| `!` | 確定読み |
| `?` | 疑問読み |
| `/` | 行分割 |
| `:` または `.` | 語分割 |
| `< >` | 書記挿入（中のテキストは保持） |
| `˹ ˺` | 半括弧（翻字から除去） |
| `[ ]` | ドキュメントレベル翻字から除去（例: `[KÙ.BABBAR]` → `KÙ.BABBAR`） |

## 前処理推奨（Replace）

以下を **置換** する:

| 元 | 変換先 |
|----|--------|
| `[x]` | `<gap>` |
| `…` | `<big_gap>` |
| `[… …]` | `<big_gap>` |
| 上付き `ki` | `{ki}`（限定詞リスト参照） |
| `il₅`（下付き数字） | `il5`（そのまま数字に） |

### Ḫ/ḫ → H/h の変換

**重要**: 学習データには `Ḫ` `ḫ` が含まれるが、テストデータには `H` `h` のみ。
アッカド語にはHは1種類しかないため、単純置換で対応可能: `Ḫ` → `H`, `ḫ` → `h`
